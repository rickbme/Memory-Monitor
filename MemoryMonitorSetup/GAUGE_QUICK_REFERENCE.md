## Gauge Details

### RAM Usage Gauge (Top Left - Blue)
- **Color**: Blue ring and needle
- **Range**: 0-100%
- **Display**: Shows "XX.X/YY" (used/total GB)
- **Updates**: Every second

### CPU Load Gauge (Top Center - Red)
- **Color**: Red ring and needle  
- **Range**: 0-100%
- **Display**: Shows percentage value
- **Updates**: Every second

### Disk I/O Gauge (Top Right - Green)
- **Color**: Green ring and needle
- **Range**: Auto-scales (10, 25, 50, 100, 250, 500+ MB/s)
- **Display**: Shows current MB/s
- **Updates**: Every second
- **Note**: Scales automatically based on peak usage

### Ethernet Gauge (Bottom Left - Yellow/Gold)
- **Color**: Yellow/Gold ring and needle
- **Range**: Auto-scales (10, 25, 50, 100, 250, 500+ Mbps)
- **Display**: Shows current Mbps
- **Updates**: Every second
- **Note**: Combined upload + download speed

### GPU VRAM Gauge (Bottom Right - Purple)
- **Color**: Purple ring and needle
- **Range**: 0 to detected GPU memory (e.g., 24 GB)
- **Display**: Shows "XX.X/YY" (used/total GB)
- **Updates**: Every second

## Menu Options

### View Menu
- **Dark Mode**: Toggle between light/dark theme
- **Show Disk Monitor**: Toggle disk gauge visibility
- **Show Network Monitor**: Toggle network gauge visibility

### Help Menu
- **About**: Display application information

## System Tray
- Application minimizes to system tray
- Tooltip shows: "CPU: XX GPU: YY"
- Double-click to restore window
- Right-click for context menu (Show/Exit)

## Gauge Features
Each gauge displays:
1. **Title** at top (e.g., "RAM USAGE", "CPU LOAD")
2. **Colored arc** showing current value with glow effect
3. **Analog needle** pointing to current value
4. **Scale numbers** around the perimeter (0 to max)
5. **Tick marks** for precise reading
6. **Digital display** at bottom with exact value
7. **Chrome rim** for professional appearance

## Performance Notes
- All gauges update simultaneously every 1000ms
- Auto-scaling prevents gauge "maxing out"
- Smooth needle animation with anti-aliasing
- Low CPU overhead with double-buffering
- Process list shows memory hogs (>400 MB RAM)

## Keyboard Shortcuts
- None currently assigned (use menu for all functions)

## Color Meanings
- **Blue**: Memory/RAM (cool, calm resource)
- **Red**: CPU (hot, high activity)
- **Green**: Disk (go/active I/O)
- **Yellow**: Network (caution/traffic)
- **Purple**: GPU (specialized/video)
